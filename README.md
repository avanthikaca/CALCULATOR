# Simple Calculator

A basic web-based calculator built using **HTML**, **CSS**, and **JavaScript**.

## 🚀 Features
- Supports basic operations: `+`, `-`, `*`, `/`
- Clear button to reset input
- Responsive layout
- User-friendly interface

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
